# 🚀 DevTRACK — Plataforma de Evolução para Desenvolvedores

> Um sistema web focado em acompanhar, organizar e acelerar a evolução profissional de desenvolvedores através de dados reais, aprendizado diário e inteligência artificial.

---

## 📌 Sobre o Projeto

**DevEvolution** é uma plataforma web que funciona como um "segundo cérebro" para desenvolvedores.

O objetivo é permitir que qualquer dev acompanhe sua evolução técnica de forma prática, conectando:

* 📚 O que aprendeu diariamente
* 💻 Atividade real no GitHub
* 📈 Progresso técnico ao longo do tempo
* 🤖 Análises inteligentes com IA

Este projeto está sendo desenvolvido como projeto acadêmico do curso de **Análise e Desenvolvimento de Sistemas (ADS)**, porém com arquitetura e visão de produto real.

---

## 🎯 Problema

Desenvolvedores estudam todos os dias, mas:

* Não conseguem medir evolução real
* Esquecem o que aprenderam
* Não possuem histórico organizado
* Não recebem feedback estruturado

O DevEvolution resolve isso centralizando aprendizado + prática + análise inteligente.

---

## 💡 Solução

A plataforma permite que o usuário:

* Registrar "Aprendi Hoje"
* Conectar conta do GitHub
* Visualizar métricas de evolução
* Receber análises geradas por IA
* Criar histórico técnico pessoal
